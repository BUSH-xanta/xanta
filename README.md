## Projects

### Nginx Access Log Parser

Security-oriented Nginx log analysis tool.

- Parses `.log` and `.gz` files
- Detects suspicious IPs, paths, User-Agents and scan-like patterns
- Exports reports to JSON, CSV and Markdown
- Supports live monitoring mode

Repository: https://github.com/BUSH-xanta/nginx-access-log-parser

### Site Contact Parser

Tool for extracting public contact data from websites.

- Extracts emails, Russian phone numbers and Telegram contacts
- Crawls main and contact-like pages
- Supports HTTPS to HTTP fallback
- Deduplicates results and exports to CSV

Repository: https://github.com/BUSH-xanta/site-contact-parser
